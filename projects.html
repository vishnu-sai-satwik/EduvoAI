Passing Criteria

The Senior AI Dev grades your code from 0 to 100. You must score at least 70/100 for your PR to be approved and earn XP. Code submitted in the wrong programming language gets an automatic 0.

📈

XP & Gamification

Successful submissions earn XP. Accumulate XP to rank up from Junior Dev to Principal. Maintaining a daily coding streak unlocks bonuses.

💾

Auto-Save & Isolation

Your code is automatically saved securely to your local browser. If you close the tab, your draft will be exactly as you left it. Your work is 100% private and isolated from other users.

🛡️

Anti-Cheat System

Massive copy-pasting, inhuman typing speeds, or frequent tab-switching will flag your session. If flagged, you must verbally defend your code logic to the AI to pass.
